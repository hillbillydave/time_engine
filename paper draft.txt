

# **Title**  
**Observer‑Dependent Information Metrics and Emergent Time in CMS D⁰ Production Data**

---

# **Abstract**  
We present a working implementation of an observer‑dependent information‑geometric time model applied directly to CMS D⁰ meson production data in ultra‑peripheral Pb–Pb collisions. Using the Observer‑Dependent Information Metric (ODIM) framework, we compute curvature, projection capacity, and collapse metrics for each kinematic bin, construct multiple observer‑specific Fisher‑type metrics, and evaluate the resulting observer‑dependent time parameters. We introduce two clock constructions — a yield‑based ODIM clock and an experimental‑style integrated‑σ clock — and compare their behavior to observer time flows. We further compute local trajectory curvature, perform multidimensional scaling (MDS) to visualize the manifold structure, and evaluate both discrete and continuous geodesics under the O2 metric. The results demonstrate strong divergence between observer time flows, stability of physical clocks, and clear geometric structure in the underlying data manifold.

---

# **1. Introduction**  
Time in this framework is not a universal scalar but an emergent property of an observer interacting with a trajectory in information space. The ODIM model defines time as the accumulated information‑distance along a path in a high‑dimensional state space derived from physical observables. This work applies the ODIM formalism to real CMS D⁰ production data, demonstrating observer‑dependent time dilation, clock stability, and geometric structure.

# **3. Observer Metrics**  
Three observers are defined via Fisher‑type diagonal metrics:

- **O1:** sensitive to curvature + projection capacity  
- **O2:** sensitive to x + Q²  
- **O3:** sensitive to σ + collapse_metric  

Each observer defines a different Riemannian metric \(g^{(O)}\).

---

# **4. Observer‑Dependent Time**  
Time for observer \(O\) is:

\[
t_O = \sum_k \sqrt{g^{(O)}_{ij}(X_k)\,\Delta X_k^i\,\Delta X_k^j}
\]

Results:

- \(t_{O1}\): ~0 → 74  
- \(t_{O2}\): ~0 → 12,674  
- \(t_{O3}\): ~0 → 40,071  

This demonstrates **massive observer‑dependent divergence**.

# **5. Clock Constructions**

### **5.1 Yield‑Based ODIM Clock**  
Clock metric emphasizes σ and collapse_metric.  
θ_yield is slow and stable — a physical chronometer.

### **5.2 Experimental‑Style Clock**  
Defined as:

\[
\theta_{\text{exp}} = \int \sigma\, ds
\]

This behaves like a real CMS clock: monotonic, physical, and insensitive to observer distortions.

### **8.2 Continuous Geodesic (Line‑Integral Approximation)**

\[

# **9. Results and Interpretation**  
- Observers age at dramatically different rates.  
- Physical clocks remain stable.  
- The trajectory has clear geometric structure.  
- Off‑geodesic steps correspond to rapid changes in σ, Q², or curvature.  
- ODIM time behaves exactly as predicted: **time is observer‑dependent.**

# **10. Conclusion**  
This work demonstrates the viability of ODIM as a physically grounded, observer‑dependent time model. Applied to real CMS data, the framework reveals strong time divergence, stable clocks, and meaningful geometric structure. Future work includes full geodesic shooting, Christoffel symbol estimation, and integration with the Light‑Wall holagraphy architecture.
